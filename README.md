# 私の学習支援システム
私は、自分の興味のあるトピックについてクイズ形式で学べる学習アプリを作りました。このアプリでは難易度を選べ手自分に合ったレベルで回答できます。
## 実装した機能
クイズを作る機能
クイズ・回答を表示する機能
正解かを判断する機能
### 基本機能
- LLMによる質問応答
- 自動生成クイズ機能

## 今後の改善予定

今後は、同じトピックについて10問のテストを行える機能も追加したいと思った。また、今回はRAGを使わずにLLMに直接渡すだけになってしまったので、RAGを活用したより良いアプリを作りたい。クイズアプリでRAGを使う場合は、多くの概念やオントロジーが必要になるので、スポーツや強化学習など特定のテーマに特化したクイズアプリで活用したいと考えている。
